# EMERGING: Multi-Regional Input–Output Table for the Global Emerging Economies

Analysis scripts for the EMERGING V2 MRIO database — 245 economies × 133 sectors, annual tables for 2015, 2018, 2021, and 2023.

## Dataset

The EMERGING database provides full-scale, near real-time multi-regional input–output (MRIO) tables covering 245 economies and 133 sectors (105 commodity + 30 service sectors). It also incorporates CO₂ emissions from fossil fuel combustion across seven energy types. All data are sourced from official national statistics to ensure consistency across regions and over time.

**Download:** [Zenodo — DOI: 10.5281/zenodo.19461860](https://doi.org/10.5281/zenodo.19461860)  
**License:** [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/)

After downloading, place the `.mat` files in a `V2/` subdirectory.

## Repository Contents

### Classification Files
| File | Description |
|------|-------------|
| `EMERGING_Country.xlsx` | Country classification for 245 economies |
| `EMERGING_Sector_V2.xlsx` | Sector classification for 133 sectors (V2) |
| `EMERGING_Sector.xlsx` | Sector classification (V1, for comparison) |
| `EMERGING_Classification.xlsx` | Combined classification output (generated by `CountrySectorClass.R`) |

### Scripts

| Script | Language | Description |
|--------|----------|-------------|
| `CountrySectorClass.R` | R | Reads country and sector labels from the raw `.mat` HDF5 files and exports them to `EMERGING_Classification.xlsx` |
| `ImportEMERGING_V2.R` | R | Imports the V2 MRIO tables, aggregates to broad sectors, and saves to `EMERGING_Broad_Sectors.qs2` |
| `STATA_ICIO_CSVs_V2.R` | R | Writes the aggregated ICIO tables from `EMERGING_Broad_Sectors.qs2` to CSV files under `ICIO_CSV/` |
| `ICIO_decomp.jl` | Julia | GVC decompositions (KWW country-level, exporter-sector, bilateral-sector) using [ICIO.jl](https://github.com/SebKrantz/ICIO.jl); writes results to `ICIO_CSV/` |
| `ICIO_decomp.do` | Stata | Stata equivalent of the GVC decompositions |
| `rvc_indicators_V2.R` | R | Computes Regional Value Chain (RVC) indicators from decomposition outputs and produces figures |
| `Compare_Classifiction_with_V1.R` | R | Compares V1 and V2 country/sector classifications |

## Workflow

```
1. CountrySectorClass.R      # extract classification from .mat files
2. ImportEMERGING_V2.R       # import & aggregate MRIO tables → EMERGING_Broad_Sectors.qs2
3. STATA_ICIO_CSVs_V2.R      # write ICIO CSVs → ICIO_CSV/
4. ICIO_decomp.jl            # GVC decompositions → ICIO_CSV/ (or ICIO_decomp.do for Stata)
5. rvc_indicators_V2.R       # RVC indicators & figures
```
